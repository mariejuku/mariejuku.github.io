Pack Description

This is a set of small mods to Trackmania 2 that try to introduce a degree of neon and movement to the base game. Many animated elements are used.

To be clear, these mods don't re-create or port Turbo textures (for that, you can go here). These are original content.

Download and install using the links below.

Important note: Due to a bug in Maniaplanet, you cannot change the dynamic time of day on a map while a mod is applied. You must return to the default mod. Once the time has been changed, you may save the map and re-apply the custom mod.

Installation

Maniaplanet / Trackmania 2:

To install, place the archive wholesale into the Mod folder within the relative environment.

The paths will be:

  • Maniaplanet/Skins/Canyon/Mod/
  • Maniaplanet/Skins/Valley/Mod/
  • Maniaplanet/Skins/Stadium/Mod/
  • Maniaplanet/Skins/Lagoon/Mod/

When updating the pack, overwrite any previous files.
